# OpenCHAMI Installer Script Development Sandbox

This repository is not intenteded for long-term use, it is a place for
me to do some development of an OpenCHAMI installation script based on
the [OpenCHAMI Hands-On Tutorial](https://github.com/OpenCHAMI/tutorial-2025)
and lifted from the
[OpenCHAMI vTDS Application Layer](https://github.com/Cray-HPE/vtds-application-openchami)
code. The intention is to tie this installer script in with the
[OpenCHAMI Release](https://github.com/OpenCHAMI/release) repository so that
there is an installer script that tracks each release of the
[OpenCHAMI Project](https://github.com/OpenCHAMI), and remove the
corresponding logic from the OpenCHAMI vTDS Application layer, since
maintaining the installation procedure there is more cumbersome and fragile.

The work is being done here until the decision to move this script into the
OpenCHAMI Release repository is finalized, at which point it will be moved
to the OpenCHAMI Release repository and discontinued (most likely deleted)
here. This repository should be treated as a temporary interim sandbox,
not an on-going project.
